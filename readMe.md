# Проект «Исторические даты»

Веб-приложение для просмотра исторических событий по категориям (Кино, Литература и др.). Реализовано круговое меню для выбора категории, слайдер событий и анимации на GSAP.

## Установка

1. Установите зависимости:

```bash
npm install
```

## Запуск

**Режим разработки** — запускает dev-сервер и открывает проект в браузере:

```bash
npm start
```

**Сборка для продакшена**:

```bash
npm run build
```

## Стек

- React 19
- TypeScript
- React Router
- GSAP
- Swiper
